# Changelog

## 1.1.0 - 2026-07-04

Initial public release.

- Track garden harvests by season, category, weight, count, and individual item weights.
- Track watering separately from harvest categories.
- Configure category icons, colors, display units, and monetary value.
- View recent harvests and interactive charts for weights, counts, watering, item distribution, and cumulative value.
- Import and export season JSON files.
- Use the UI in English, Finnish, Swedish, German, French, or Spanish.
- Install on Linux with a Debian package.
