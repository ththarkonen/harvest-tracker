# Changelog

## 1.1.3 - 2026-07-04

- Show the MIT license in the installer-facing app description as a fallback for graphical installers that omit the license tile for local Debian packages.
- Use a normal Debian package synopsis and long description for graphical installer metadata.

## 1.1.2 - 2026-07-04

- Add AppStream metadata so graphical software installers can show the project website and app details.
- Include Debian copyright metadata so graphical software installers can show the MIT license.
- Keep packaged chart views legend-free to match the development view.

## 1.1.1 - 2026-07-04

- Make Plotly chart markers consistent across line-based charts.
- Improve chart resizing so plot views use the available panel height.
- Restore the desktop View menu with standard zoom controls.

## 1.1.0 - 2026-07-04

Initial public release.

- Track garden harvests by season, category, weight, count, and individual item weights.
- Track watering separately from harvest categories.
- Configure category icons, colors, display units, and monetary value.
- View recent harvests and interactive charts for weights, counts, watering, item distribution, and cumulative value.
- Import and export season JSON files.
- Use the UI in English, Finnish, Swedish, German, French, or Spanish.
- Install on Linux with a Debian package.
